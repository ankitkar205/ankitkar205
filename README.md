### 🚀 About Me

I am a passionate developer and hardware enthusiast focused on building innovative solutions that make a real difference. I blend software engineering with hardware design to create everything from automation scripts to life-changing assistive devices.

*   🔭 **Currently Building:** Assistive technologies including a **self-stabilizing spoon for Parkinson’s patients**, a smart plate stand, a mental health analysis system, and various voice-assistant automation tools.
*   🤝 **Looking to Collaborate On:** Projects involving Embedded Systems, Microcontrollers, Cybersecurity, Biotech, EEG, Machine Learning, and tech for social impact.
*   🌱 **Currently Learning:** Advanced Python & Java, Machine Learning algorithms, Linux environments, and secure embedded system design.
*   💡 **Ask Me About:** Arduino, C/C++, IoT setups, Assistive Devices, and creative post-production (Photoshop/Premiere Pro).
*   ⚡ **Fun Fact:** I balance logical engineering with artistic creativity—when I'm not wiring hardware or writing code, I'm working on professional photo and video editing.
